## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YashRajveerSingh/My-Portfolio.git
```

Move to the project directory:

```bash
cd My-Portfolio
```

Install dependencies:

```bash
npm install
```

Create environment file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Start development server:

```bash
npm run dev
```

---

## 🏗️ Production Build

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
